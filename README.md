# AI 文档翻译助手 (AI Document Translator)

一款支持 `EPUB` 和 `DOCX` 格式的文档翻译工具。应用底层结构解析与“锚点跟随”技术，确保译后文档的样式、图片及排版结构与原文档一致。

---

## �️ 技术机制 (Mechanisms)

### 1. 处理流程
程序采用 **“提取 - 锚点化 - 镜像同步”** 的处理逻辑。

### 2. 核心技术
*   **锚点跟随 (Anchor-Following)**：
    将文档中的格式标签（如 `<b>`, `<img>`, `<a>`）提取并替换为唯一字符锚点（如 `⭀`, `⭁`）。AI 仅处理纯文本，翻译完成后程序根据索引号还原标签位置。
*   **实时源码镜像 (Live Source Mirror)**：
    在缓存目录维护一套解压后的源码镜像。每个分块翻译完成后即时回写至镜像文件。该机制确保了进度的持久化，即使程序中断，已翻译内容也已写入文件。
*   **ID 同步 (ID Synchronization)**：
    在 EPUB 的 HTML 标签中注入 `data-trans-idx` 属性。翻译进度直接与文档结构绑定，提高增量更新与断点续传的稳定性。
*   **智能边界分块 (Intelligent Chunking)**：
    根据字符限制自动合并段落，并严格遵循文件边界（File Boundary），确保 AI 具备完整的上下文且不跨文件操作。

## 🎮 功能与使用方法

### 1. 配置区域 (顶部)
*   **文件选择**：加载 `.epub` 或 `.docx` 文件。
*   **API 配置**：
    *   **Endpoint / Key**：兼容 OpenAI 格式的接口。
    *   **Model**：指定使用的 LLM 模型。
    *   **分块字符 (Chunk Size)**：单次任务的字符量，建议 **1500-2500**。
    *   **上下文轮数 (Context)**：参考历史翻译的轮数，建议设置为 **1**。

### 2. 任务管理 (左侧列表)
*   **初始化**：点击“分块并分组”解析全书。
*   **状态指示**：
    *   ⚪ **灰色**：未处理。
    *   🔵 **蓝色**：处理中。
    *   🟢 **绿色**：成功。
    *   🔴 **红色**：失败或格式错误（自动回退至原文）。
*   **操作**：支持多选（Ctrl/Shift）后进行局部翻译。

### 3. 编辑预览 (右侧面板)
*   **双语对照**：左侧原文，右侧译文。
*   **手动干预**：可直接在右侧编辑框修改译文，保存后即时同步至镜像。
*   **辅助工具**：内置行号同步，方便定位。

## 🚀 运行与构建

### 1. 直接运行 (Release)
1. 下载并解压 Release 版本。
2. 双击 **`run.bat`** 启动。

### 2. 源码运行 (Source)
1. 安装 Python 3.10+。
2. `pip install -r requirements.txt`
3. `python main.py`

### 3. 自行构建 (Build)
*   **Windows**：运行 `build_exe.bat`。
*   **macOS**：运行 `build_macos.sh`。
*   产出物位于 `dist/` 目录。

## 🤖 提示词建议 (Prompt)

**推荐提示词：**
```text
你是一个专业的书籍翻译引擎。你的任务是将输入的文本块从【源语言】翻译为【目标语言】。

规则：
1. **保留锚点**：严格保留所有特殊成对符号（如 `⩀`, `⩁` 等标识块颜色，`⭀`, `⭁` 等标识块内标签），它们是结构锚点。
2. **块级对应**：每个 `⩀...⩀` 对应一个翻译块，数量和顺序必须与原文完全一致。
3. **标签保护**：保持标签符号（如 `⭀文字⭀`）在块内的相对位置，仅翻译其中的文字。
4. **简洁输出**：直接输出翻译内容，不要有任何额外解释。

示例：
输入：⩀Hello ⭀world⭀!⩀
输出：⩀你好 ⭀世界⭀！⩀
```

## ⚠️ 注意事项与常见问题

### 1. 注意事项
*   **文件占用**：操作文件前请关闭 Word、Calibre 等阅读器，防止由于权限冲突导致读写失败。
*   **锚点保护**：不要删除或修改译文中的特殊符号（如 `⩀`, `⭀`），否则将导致该处排版失效。
*   **长度限制**：若单段超长（>4000 字符），可能触发 AI 响应截断，建议调小分块大小。

### 2. 可能存在的问题
*   **API 异常**：网络或服务商问题可能导致超时。建议调整上下文轮数或更换 API。
*   **格式不匹配**：若 AI 合并了多个句子导致锚点丢失，程序会将该块标红。此时需手动重新处理。
*   **格式转换**：暂不支持 PDF。建议先转换为 DOCX 后再进行翻译。
